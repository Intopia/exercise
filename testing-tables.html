Example 2

This example uses a <caption> to label the <table> and <th> for column and row headers.

scope="col" is used to associate header cells with the columns below them. scope="row" is used to associate header cells with the rows beside them.

scope explicitly identifies whether a header applies to a row or a column. For most simple tables, browsers can infer these relationships automatically, so they work correctly without scope. Adding scope simply makes the author's intent explicit.
