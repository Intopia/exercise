Testing image types

Type 1: Decorative image

Decorating image using an empty alt attribute:

<img src="decorative.jpg" alt="">

Type 2: Informative images

Informative image with a non-empty alt attribute:

<img src="informative.png" alt="Sydney Opera
House Environmental Sustainability Manager
Emma Bombonato, with some of the 35 kilometres
of pipes that heat and cool the building">

Type 3: Functional images

Functional image with alt attribute applied to the <img> element:

<button>
  <img src="functional.png" alt="Close">
</button>

Functional image with aria-label attribute applied to the <button> element:

<button aria-label="Next">
  <img src="functional.png" alt="">
</button>

This is not a functional image as there is text available that describes the function:

<button>
  <img src="functional.png" alt=""> Dismiss
</button>

Type 4: Complex images

Complex image example with alt attribute and longer descrition below:

<img aria-details="aaa" src="complex.png"
alt="Venn diagram depicting content category
relationships. Detailed description below">
