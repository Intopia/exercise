Installing browser extensions and add-ons

A browser extension is a small software add-on for your internet browser, such as Chrome, Firefox, Edge, or Safari. It enhances your browsing experience by adding extra features or functionalities, such as blocking ads, managing cookies, or customizing the look of websites. This makes your online experience more personalized and convenient.

For this course, we will use two browser extensions that can be used to test various aspects of accessibility:

  • Axe DevTools
  • Accessibility Insights for Web

How to install the extensions in Chrome 

  1. Open Chrome and go to the Chrome Web Store links.
    1. Axe Devtools
    2. Accessibility Insights for the web
  2. Click "Add to Chrome."
  3. Confirm by clicking "Add extension."

How to install the add-ons in Edge 

  1. Open Edge and go to the Microsoft Edge Add-ons store.
    1. Axe Devtools
  2. Click "Get" and then "Add extension."
  3. Confirm the installation.

How to install the add-on in FireFox

  1. Open Firefox and go to the Firefox Add-ons links.
    1. Axe Devtools
    2. Accessibility Insights for the web
  2. Click "Add to Firefox."
  3. Confirm by clicking "Add."
