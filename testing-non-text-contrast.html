Testing non-text contrast

WCAG 2.1 requires a contrast ratio of at least 3:1 for graphics and user interface components (such as form input borders).

In the examples below, we need to measure the contrast between the input borders and the surrounding colour - in this case the background-colour - which has a HEX value of #edebeb.
