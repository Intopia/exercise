All form controls are defined using the required attribute. In Chrome's accessibility tree, all form controls:

  • Should be defined as: Required: true.
  • Should be defined as: Invalid user entry: false.

  • PASS: Registers the required attribute.
  • PASS Registers as Invalid user entry: false.
